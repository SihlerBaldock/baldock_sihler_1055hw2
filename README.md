# Multimedia Authoring 2 - HW #2 - Crushing Bugs
The goal of this repository is to find two bugs on a simple drag-and-drop game on a webage. The first bug involves only allowing one dragged element into each dropzone, because currently multiple dragged elements can be put into the dropzones. The second bug involves adding functionality to the reset button, because currently nothing happens. The reset button should send the dragged elements back to where they started.
## Installation
No installation required. Simply download the zip file and extract.

## Usage
Open index.html in the browser of your choice.
